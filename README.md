# bruno-nog-eduick

## Instalação e execução
1. Clone o repositório:
```git clone git@github.com:BrunoNogueiraBruno/bruno-nog-eduick.git```

2. Navegue para a pasta principal do projeto:
```cd bruno-nog-eduick```

3. Instale as dependências:
```npm install```

4. Execute o projeto:
```npm run dev```
